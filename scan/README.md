公司服务器的规模逐渐成规模化后，有一个比较严肃的安全问题：所有服务器的端口开放情况及其协议。

1. 归档服务器的端口及其协议，用于统计与维护
2. 发生新的安全漏洞时，可以根据协议，定位到具体的服务器
3. 加强对服务器的管理，只允许开放规定的端口

在本系列中，我们将从需求确认，方案选择，具体实现环节来阐述具体实现。

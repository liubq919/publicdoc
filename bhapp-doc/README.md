### 堡垒机申请

    在各公司中，资源申请，一直是一个人工参与度高，流程繁琐，审核慢的话题。堡垒机申请，就是其中的一个案例。
    1. 由于流程或者堡垒机的差异，一个简单的需求，都可能变的十分繁琐
    2. 申请的资源在创建成功后，需要完成后续的通知与定期检查
    3. 整合数据，对管理提供数据支持
